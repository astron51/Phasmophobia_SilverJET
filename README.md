# Phasmophobia_SilverJET
Cheat made for Phasmophobia

Basic function :
- Ghost, Bone, Ouija Board, EMF Interaction and Fusebox ESP
- Ghost Activity Graph
- Mission Information including ghost's name, age and etc...
- Ghost Box
- ~~Phrase Recognizer~~
- ~~Adjustable Flashlight~~
